PeakPass

Nov. 1st 2022 ● GitHub

PeakPass is a password manager that also doubles as a tool to check your passwords against those that have been affected in previous data breaches. Along with this we offer a blog to teach users about online standards that help to keep them safe when using the internet.

This is our submission that won 2nd place in the 2022 Congressional App Challenge Competition. It has been open sourced for transparency.

EduStore

Nov. 1st 2023 ● GitHub

EduStore looks to fill a gap in any School District's extensive inventory system. Currently implemented in Mansfield ISD's Technology Department, our program allows this 54 campus school district to easily manage the massive number of items in inventory and get alerted when essential items become low on count.

Furthermore, EduStore allows admins to keep track of which users are taking out items and constantly observe the number of all parts currently in inventory. Lastly, we've created functionality for school district's to create administrator accounts to add new items to inventory based on their Purchase Order.

This is our submission for the 2023 Congressional App Challenge Competition. It has been open sourced for transparency.
